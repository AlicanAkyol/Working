To run the project;
Terminal opens
* Go to the project's directory
* run command -> node chainway.js
* Postman application opens
* A post request is created in postman.
* In the postman post request, enter http://0.0.0.0:3001/new-tx in the url section.
* In Postman, select the raw option in the body section and add the relevant txid; Orn: {
    "txid": "11d53c83795205b512c92b142ce845e497b5724e3482498c3000ad29e2e23cc5"
}
